This is the Grocery Stored, which is built using Flask and sqlite3.
It is a multi-user app(one required admin/store manager and other users). Used for buying and selling groceries. User can buy many products for one or multiple sections. Store manager can add section/category and products.

To run this,
first check for 'my_database.db'
  ->if it is not present then first run "python sqlite.py" to create database
  ->Then run 'main.py'
If 'my_database.db' is present :
  ->Then directly run 'main.py'
